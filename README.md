A sample command-line application with an entrypoint in `bin/`, library code
in `lib/`, and example unit test in `test/`.

1. Hitung Jumlah Angka Genap
- Deskripsi:
Buat fungsi untuk menghitung berapa banyak bilangan genap di dalam sebuah array integer.
- Contoh Input:
array = [2, 5, 8, 11, 14]
 
2. Balik Urutan Kata
- Deskripsi:
Buat fungsi untuk membalik urutan kata dalam sebuah kalimat, tanpa membalik huruf per kata.
- Contoh Input:
“Hello World”

Catatan: Kode Ada di Folder bin dan juga test untuk skenario unit test

Untuk menjalankannya di terminal gunakan perintah:
- dart run bin/soal_pertama.dart
- dart run bin/soal_kedua.dart
